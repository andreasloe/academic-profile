Software and Data Contributions

  • The R statistical software uses work from Andreas Loeffler for the Wilcoxon distribution implementation. The R source file src/nmath/wilcox.c states: “The Wilcoxon distribution is calculated using work from Andreas Loeffler” and links to the 1984 note on partitions and the Wilcoxon-Mann-Whitney test. Sources: R Bugzilla issue 18655 and the R source file src/nmath/wilcox.c; note: PDF.
