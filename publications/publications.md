## Software and Data Contributions

- The R statistical software uses work from Andreas Loeffler for the Wilcoxon distribution implementation. The R source file `src/nmath/wilcox.c` states: "The Wilcoxon distribution is calculated using work from Andreas Loeffler" and links to the 1984 note on partitions and the Wilcoxon-Mann-Whitney test. Sources: R Bugzilla issue [18655](https://bugs.r-project.org/show_bug.cgi?id=18655) and the R source file [`src/nmath/wilcox.c`](https://raw.githubusercontent.com/wch/r-source/trunk/src/nmath/wilcox.c); note: [local PDF](../assets/pdfs/LoefflerWilcoxonMannWhitneyTest.pdf), [source PDF](https://upload.wikimedia.org/wikipedia/commons/f/f5/LoefflerWilcoxonMannWhitneyTest.pdf).
