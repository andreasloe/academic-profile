# academic-profile

Academic profile website — publications, CV, and contact information.

## Structure

| File | Purpose |
|------|---------|
| `index.html` | Main profile page (About, Publications, CV, Contact) |
| `style.css` | Stylesheet for the profile page |
| `cv.pdf` | Curriculum vitae *(add your own)* |

## Getting started

1. Edit `index.html` to fill in your name, position, institution, research summary, and publications.
2. Replace the `cv.pdf` placeholder link with your actual CV file.
3. Update the contact details and links as needed.
4. Open `index.html` in a browser to preview, or deploy to any static hosting service (GitHub Pages, Netlify, etc.).
